WHAT IS A PSEUDO ELEMENT?

A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected element(s). For example, ::selection can be used to change the style of the selected text. Select this paragraph to see the CSS in action.


CSS

      p {
        font-family: 'Source Code Pro', monospace;
        text-shadow: 3px 3px 11px rgba(93, 107, 96, .5);
        font-size: 20px;
      }

      p::selection {
        color: rgb(236, 226, 246);
        background: rgb(254, 169, 99);
        text-shadow: 3px 3px 11px rgba(255, 255, 255, 0.9);
      }

      b {
        color: rgb(255, 255, 255);
        font-size: 24px;
      }


      b::selection {
        color: rgb(231, 167, 162);
        background: rgb(195, 162, 228);
        text-shadow: none;
      }

WHAT IS A PSEUDO CLASS?

A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s). For example, :hover can be used to change a button's color when the user hovers over it. The ::before pseudo element is the first child of the selected element. It usually is added for design purposes to an element with the content property and is inline by default.

      li.done::before {
        content: '';
        position: absolute;
        border-color: rgba(232, 232, 232, 0.49);
        border-style: solid;
        border-width: 0 0.3em 0.25em 0;
        height: 1em;
        top: 1.3em;

        margin-top: -1em;
        transform: rotate(45deg);
        width: 0.5em;
      }

      li:hover {
        background: rgba(204, 204, 204, 0.42);
      }

JavaScript

      var list = document.querySelector('ul');
      list.addEventListener('click', function(e) {
        if( e.target.tagName === 'LI') {
           e.target.classList.toggle('done');
        }
      }, false);
